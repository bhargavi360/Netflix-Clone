Netflix Clone – Frontend UI Project
A Netflix-inspired responsive homepage built using HTML and CSS.
This project replicates the core UI layout of Netflix including navigation bar, hero video section, scrollable movie rows, and footer.

Project Overview
Situation
As a B.Tech student focusing on frontend development, I wanted to strengthen my UI design and layout skills by recreating a real-world streaming platform interface.

Task
The goal was to design and develop a Netflix homepage clone that includes:

Responsive navigation bar
Hero section with background video
Scrollable movie posters section
Interactive hover effects
Professional footer layout
Clean and structured UI design
Action
To achieve this:

Structured the webpage using semantic HTML
Used CSS Flexbox for alignment and layout
Implemented background video with overlay effect
Added horizontal scrolling with overflow-x
Designed hover animations for movie posters
Created a responsive footer section
Organized project using proper folder structure
Version controlled the project using Git and GitHub
Result
Successfully built a visually appealing Netflix-style homepage
Improved understanding of layout design and positioning
Strengthened CSS concepts like Flexbox and overlays
Uploaded and maintained project on GitHub
Prepared foundation for adding JavaScript and backend integration in future
